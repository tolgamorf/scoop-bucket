# env2op-cli Scoop Bucket

Scoop bucket for [env2op-cli](https://github.com/tolgamorf/env2op-cli) - Push .env files to 1Password and pull them back.

## Installation

```powershell
# Add this bucket
scoop bucket add env2op-cli https://github.com/tolgamorf/scoop-bucket

# Install env2op-cli
scoop install env2op-cli
```

## Updating

```powershell
scoop update env2op-cli
```

## Available Apps

| App | Description |
|-----|-------------|
| [env2op-cli](bucket/env2op-cli.json) | Push .env files to 1Password and pull them back |
