# env2op Scoop Bucket

Scoop bucket for [env2op-cli](https://github.com/tolgamorf/env2op-cli) - Push .env files to 1Password and pull them back.

## Installation

```powershell
# Add this bucket
scoop bucket add env2op https://github.com/tolgamorf/scoop-bucket

# Install env2op
scoop install env2op
```

## Updating

```powershell
scoop update env2op
```

## Available Apps

| App | Description |
|-----|-------------|
| [env2op](bucket/env2op.json) | Push .env files to 1Password and pull them back |
